# React Monorepo

a simple monorepo setup for `react` & `react-native` using `yarn workspaces`.

![](./docs/demo.gif)

## Getting Started

```sh
# root folder
yarn

# to run native
cd packages/native
yarn ios
yarn android

# to run web
cd packages/web
yarn start
```
